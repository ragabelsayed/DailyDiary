# DailyDiary
 A complete offline app for dailyDiary writing.
## Fearures:
- [x] CRUD operations.  
- [x] (Gallery & Camera Image, CustomColor) Cover.  
- [x] Lock Code to secure or close(Diary, Chapter, Page).  
- [x] Dark and Light Themes.  
- [x] local storge.  
- [x] Animation.  
- [x] Writing feature:
    * More Fonts.
    * Change Color.
    * Text Alignment.
    * Bold and Italic Text.
